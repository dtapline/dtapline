Add automatic diff URL generation for deployments. Diff URLs show code changes between sequential environments (e.g., dev → staging → production) and are auto-generated based on project's repository URL. Supports GitHub, GitLab, Bitbucket, and Azure DevOps. Also improved UI display of repository URLs in service overview.
